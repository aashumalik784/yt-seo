# 🚀 YouTube SEO AI Booster

AI-powered YouTube SEO tool jo aapki videos ko organic tarike se boost karta hai.

## ✨ Features
- 🤖 AI-generated Titles, Description, Tags (Hindi/English/Hinglish)
- 📈 Live Video Analytics Graph (Views/Likes tracking)
- 🔥 Trending Topics Suggestions
- 🎯 SEO Score Checker

## 🛠️ Tech Stack
- **Frontend:** React + Vite + Tailwind CSS
- **Backend:** Cloudflare Workers
- **AI:** Google Gemini API
- **Hosting:** Cloudflare Pages + Workers
