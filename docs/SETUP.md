# 🛠️ Setup Guide

## Prerequisites
- Node.js 18+
- npm or yarn
- Cloudflare account
- Google Gemini API key
- YouTube Data API v3 key

## Installation

### 1. Clone the repo
```bash
git clone https://github.com/YOUR_USERNAME/yt-seo-ai.git
cd yt-seo-ai
```

### 2. Install all dependencies
```bash
npm run install:all
```
